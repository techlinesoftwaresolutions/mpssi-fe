# MPSSIWeb-FrontEnd

A modern, responsive website for Maharishi Prajapati Shiksha Samiti - dedicated to education, social welfare, and community empowerment.

## Features

- Responsive design with Tailwind CSS
- Smooth animations with Framer Motion
- Bilingual content (Hindi & English)
- Event management and gallery
- Patron recognition system
- Scholarship application portal
- Committee history tracking

## Getting Started

\`\`\`bash
npm install
npm run dev
\`\`\`

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
